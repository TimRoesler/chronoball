│     1 # Chronoball                                                                                            │
│     2                                                                                                         │
│     3 Chronoball is a dynamic Foundry VTT minigame module for turn-based ball competitions. Featuring two     │
│       teams, strategic phases, comprehensive scoring, and an intuitive HUD, it brings exciting grid-based     │
│       sports action to your virtual tabletop.                                                                 │
│     4                                                                                                         │
│     5 ## Features                                                                                             │
│     6                                                                                                         │
│     7 *   **Team Management:** Easily create and manage two competing teams, "Team A" and "Team B".           │
│     8 *   **Turn-Based Gameplay:** The game is structured in turns, allowing for strategic decisions.         │
│     9 *   **Player Actions:** Players can perform various actions like moving, carrying the ball, and         │
│       throwing the ball.                                                                                      │
│    10 *   **Scoring System:** Score points by carrying or throwing the "Chronoball" into the opponent's       │
│       endzone.                                                                                                │
│    11 *   **Interactive HUD:** A heads-up display provides real-time information about the game state         │
│       (current turn, score, etc.).                                                                            │
│    12 *   **Customizable Rules:** Configure game mechanics like grid size, movement points, and more through  │
│       the Rules Panel.                                                                                        │
│    13 *   **Visual Effects:** Integration with the "Sequencer" module provides visual effects for actions     │
│       like throwing the ball.                                                                                 │
│    14                                                                                                         │
│    15 ## Installation                                                                                         │
│    16                                                                                                         │
│    17 1.  Go to the "Add-on Modules" tab in your Foundry VTT game world.                                      │
│    18 2.  Click "Install Module".                                                                             │
│    19 3.  In the "Manifest URL" field, paste the following URL:                                               │
│    20     ```                                                                                                 │
│    21     https://github.com/TimRoesler/chronoball/releases/latest/download/module.json                       │
│    22     ```                                                                                                 │
│    23 4.  Click "Install" and wait for the installation to complete.                                          │
│    24 5.  Enable the "Chronoball" module in your world's module settings.                                     │
│    25                                                                                                         │
│    26 ## How to Play                                                                                          │
│    27                                                                                                         │
│    28 1.  As the Game Master, open the Chronoball Player Panel from the scene controls.                       │
│    29 2.  Use the Roster tab to assign tokens to Team A and Team B.                                           │
│    30 3.  Use the Rules Panel to configure the game settings.                                                 │
│    31 4.  Start the game! Players can use the Player Panel to perform actions on their turn.                  │
│    32 5.  The HUD will show the current game state. The GM can manually adjust the score and game phase if    │
│       needed.                                                                                                 │
│    33                                                                                                         │
